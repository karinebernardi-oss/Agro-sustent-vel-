# Agro Sustentável 

##   Objetivo 
Incentivar práticas agrícolas que preservem o meio ambiente e aumentam a proodutividade. 

## Ações
- Economia da água
- Rotação de culturas
- Compostagem
- Preservação do solo
- Uso de tecnologias systentáveis
  Beneficios
  - Menor impacto ambiental
  - Maior qualidade do solo
  - Prodção eficiente
  - Conservação dos recursos naturais
